Update Duty Desires


Select your scheduling preference for each duty below.

If you select 'Normally' then the scheduling program is free to assign you that particular duty as needed. You will never be assigned a duty you must prepare for two weeks in a row. You will never be assigned the same duty more often than 3, 4, or 5 weeks apart (depending on the duty).

If you select 'Only if absolutely necessary' then you will only be assigned if no one else who selected 'Normally' is available.

If you select 'Never' then you will never be assigned that duty.
